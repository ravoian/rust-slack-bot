# Rust Slack Bot

Sample setup for posting a message to Slack using Rust

## For running locally:

1. Install Rust [Rust Tools](https://www.rust-lang.org/tools/install)

2. Configure the necessary Slack values for token, channel, etc.

3. Run `setup.bat`
